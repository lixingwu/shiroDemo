### 菜单表

`数据库表：sys_menu`
```text
    Field        Type                Comment                                       
    -----------  ------------------- ----------------------------
    id           bigint(11)          菜单编号                                  
    parent_id    bigint(11)          菜单的父级，0表示根菜单            
    menu_type    int(11)             菜单类型，1横向菜单，2纵向菜单  
    menu_title   varchar(60)         菜单标题（20字内）                    
    menu_note    varchar(255)        菜单副标题（85字内）                 
    menu_url     varchar(255)        菜单链接地址                            
    menu_class   varchar(255)        菜单的样式                               
    create_time  datetime            菜单创建时间                            
    status       int(11)             菜单的状态                               
    sort         decimal(16,2)       排序                                        
```

`说明：`
> 1. 菜单表用于记录菜单集合，用parent_id表示父级菜单的编号，0表示为根菜单；
> 2. 菜单和角色绑定，使用中间表给角色绑定多个菜单，实现不同角色显示不同的菜单；
